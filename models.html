Trees and palms

Local height models

We generated local height models from 2802 tree individuals and 122 palm individuals.

With the generated models we predict the height for 55516 tree individuals (95.2%) and 5499 palm individuals (97.83%)

We used four different approaches for generating the models: \[ log 1: log(H) = a+ b*log(D) \] \[ log 2: log(H) = a+ b*log(D) + c*log(D)^2 \] \[ weibull: H = a*(1-exp(-(D/b)^c)) \] \[ michaelis: H = (A * D)/(B + D) \] Best model for trees and palms according to the RSE and the AIC value was log 2.

Above ground biomass (AGB)

Using the estimated height (H) for the individuals that wasn’t measure in the field, their DBH and the wood density (WD) we estimate the above ground biomass (AGB). \[ AGB = 0.0673 * (WD * H * DBH^2)^{0.976} \]
